# 📨 Telegram-бот рассылок

Бот для управления рассылками в Telegram-группах с системой ролей ROOT/OWNER и белым списком групп.

---

## 🚀 Быстрый старт

### 1. Установить зависимости

```bash
pip install -r requirements.txt
```

### 2. Настроить переменные окружения

Создайте файл `.env` или задайте переменные окружения:

```bash
export BOT_TOKEN="ВАШ_ТОКЕН_ОТ_BOTFATHER"
export ROOT_ID="ВАШ_TELEGRAM_ID"
```

Либо отредактируйте `config.py` напрямую.

**Как узнать свой Telegram ID?** Напишите боту [@userinfobot](https://t.me/userinfobot).

### 3. Запустить бота

```bash
python bot.py
```

## ⚙️ Функционал

### Роли
| Роль  | Описание |
|-------|----------|
| ROOT  | Задаётся через `ROOT_ID` в `config.py`. Полный доступ. |
| OWNER | Добавляется ROOT'ом. Может создавать/удалять рассылки. |

### Главное меню
- **📋 Админ панель** — управление овнерами и группами (только ROOT)
- **📨 Создать рассылку** — пошаговое создание (ROOT + OWNER)
- **📑 Мои рассылки** — список рассылок с просмотром, редактированием, удалением
- **📊 Логи рассылок** — журнал отправок и удалений
- **ℹ️ Info** — информация о группе/пользователе

### Рассылки
- Создаются пошагово: текст → часы (МСК) → название
- Часы формат: `9,12,15,18` (отправка в 09:00, 12:00, 15:00, 18:00 МСК)
- Отправляются во **все группы белого списка**
- Расписание восстанавливается после перезапуска

### Команда /info
Работает в любом чате. Показывает ID группы, есть ли она в белом списке, список овнеров.

## 🖥 Запуск на bothost.ru

```bash
# Установить Python 3.11 (если не установлен)
python3.11 -m pip install -r requirements.txt
python3.11 bot.py
```

Для автозапуска используйте встроенный менеджер процессов bothost.ru или добавьте в cron:
```bash
@reboot cd /path/to/tg_broadcast_bot && python3.11 bot.py >> logs/bot.log 2>&1
```
